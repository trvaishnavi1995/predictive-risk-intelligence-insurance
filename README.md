# Predictive Risk Intelligence & Claims Analytics in Workers’ Compensation Insurance

## Project Overview
An end-to-end insurance analytics project designed to explore Workers’ Compensation claims data through data cleaning, exploratory analysis, visualization, and risk intelligence modeling. The project aims to identify claim trends, underwriting exposure, industry risk concentration, and predictive risk indicators within a real-world P&C insurance environment.

Using Python-based analytics tools, the project demonstrates practical experience in handling large-scale, unstructured insurance datasets while generating meaningful business insights for underwriting and risk management

---

## Objectives
- Perform end-to-end data cleaning and preprocessing
- Analyze claims patterns and risk trends
- Identify high-risk industries and exposure categories
- Generate business insights using statistical analysis
- Build visual analytics dashboards using Python
- Create predictive risk intelligence indicators

Dataset Features:

- Industry classifications

- Carrier information
- Claim identifiers
- Geographic information

## Dataset Filtering

The original Workers’ Compensation claims dataset contained records beginning from the year 2000. 

For this project, the dataset was filtered to include claims data from 2010 to 2026 in order to:
- Focus on modern insurance claim trends
- Improve consistency in reporting formats
- Reduce outdated or incomplete historical records
- Enhance the relevance of predictive risk analysis
- Improve computational efficiency for large-scale exploratory data analysis
- Claim types and policy-related variables
